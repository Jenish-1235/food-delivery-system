# Food Delivery Application

## Overview
This repository contains backend example for Food Delivery Application like Zomato, Swiggy etc.
We aim to reproduce the closest result and this repository is built for educative purposes only and we remain committed to keep the code open and extensible as part of FOSS community.

# Development Setup
1. Copy `.env.example` to `.env` and update values
2. Run `docker-compose up -d`
3. Access:
   - Backend: http://localhost:8080
   - Prometheus: http://localhost:9090
   - Grafana: http://localhost:3000
